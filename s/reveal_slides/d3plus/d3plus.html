D3: Data Driven Documents

  • Javascript visualization library
  • Web standards
  • Michael Bostock, Vadim Ogievetsky, Jeffrey Heer
  • Stanford / University of Washington
  • 2011 -> Present
  • Very powerful
  • Steep learning curve
  • Idiosyncratic data structures

D3 Integration

  • Integrating a D3 visualization:
  • Copy JS / CSS
  • Hardcode data
  • Verify visualization works
  • Write (huge) formatData function
  • Unintuitive
  • “Hacky”

D3Plus

  • Builds on top of D3
  • Alexander Simoes & Dave Landry
  • Simplified data structures
  • Simplified API
  • Same data structure as Splunk
  • "Data visualization made easy"
